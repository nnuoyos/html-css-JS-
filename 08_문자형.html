자료형 : 문자형

  • 문자는 반드시 따옴표를 사용할 것
  • 큰따옴표 또는 작은따옴표중 어떤것을 사용해도 상관이 없다
  • 따옴표 안에 또다른 따옴표가 필요할 때에는 반대되는 따옴표를 사용한다
    '오늘은 "수요일" 입니다' "오늘은 '수요일' 입니다"
  • 이스케이프 시퀀스
    큰따옴표 안에 큰따옴표, 작은따옴표 안에 작은따옴표를 사용하기 위해서
    사용하고자하는 각 따옴표 앞에 백슬래쉬 \(키보드 원화 표시)를 붙여서 사용이 가능하다
    문자로 인식해야하는 따옴표 앞에 백슬래쉬를 사용
    '오늘은 \'수요일\' 입니다'
  • 위와 같은 방법으로 사용하면 문장도 길어지고, 따옴표가 많이 사용되었을때 코드가 지저분하다
    그래서 백틱(` :키보드 물결표시)을 사용할 수 있다. 백틱안에서 사용되는 따옴표는 모두 문자로 인식한다
    '오늘은 '수요일' 입니다' `
    "오늘은 "수요일" 입니다" `
  • 문자는 더하기 연산자에 의해 결합될 수 있다
    '오늘은' + '수요일' + '입니다' =>오늘은수요일입니다
  • 따옴표안의 내용은 문자형을 나타내기 때문에 '2'는 문자로 인식 된다
    2 + '2' => 22 출력
  • 2는 숫자형이고 '2'는 문자형이므로 연산이 안되어야 하지만, 자바스크립트의 자동형변환 규칙에 의해
    자동으로 형을 변환 시켜주기 때문에 연산이 이루어진다
    숫자로 인식되어지는경우 계산을 진행하나, 더하기 연산의 경우 숫자 더하기보다 문자더하기가 우선순위를 가진다 따라서 22가 출력된다
