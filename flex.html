display: felx; 정렬을 위한 속성값

  1. display : flex; 를 정렬하고자하는 요소들의 부모요소에 넣어준다(부모를 띄어 넘어 더 상위요소에는 안됨)
  2. flex 된 부모요소에 정렬속성들을같이 넣어줘야 한다

  3. flex-direction:나열 방향 지정 속성

  4. row : 요소들의 왼쪽에서 오른쪽으로 정렬(기본값)

    row-reverse : 요소들을 오른쪽에서 왼쪽으로 정렬

    column: 요소들을 위-아래로 정렬

    column-reverse: 요소들을 아래-위로 정렬

    5. 예시
    • menu1
    • menu2
    • menu3
    • menu4
    • menu5
    • menu6

  5. justify-content : 위치와 간격 좆러 속성

    flex-start : 요소들을 시작지점으로 정렬

    flex-end: 요소들 끝지점으로 정렬

    cener:가운데 정렬

    space-between: 지정된 너비 안에서 요소들 사이에 동일한 간격

    space-around : 지정된 너비 안에서 요소들 양옆으로 동일한 간격

    flex-direction : column;이 오면 세로정렬 속성이 됨 / row가 오면 가로정렬 속성 됨

  6. 예시
    • menu1
    • menu2
    • menu3
    • menu4
    • menu5
    • menu6

  7. align-items : 위치 조절 속성

    flex-start : 요소들을 시작지점으로 정렬

    flex-end: 요소들 끝지점으로 정렬

    cener:가운데 정렬

    baseline : 요소들을 시작위치에 정렬

    stretch : 요소들 컨테이너에 맞도록 늘림 상위요소에 높이값이 있다면, 요소가 높이값 만큼 쭉 들어오게 하는것

    flex-direction : column;이 오면 가로정렬 속성이 됨 / row가 오면 세로정렬 속성 됨

  8. 예시
    • menu1
    • menu2
    • menu3
    • menu4
    • menu5
    • menu6

  9. flex-wrap : 요소의 감싸기 속성 / 한줄로 작성할지 여러줄 작성할지 선택해줌

    nowrap: 모든 요소들을 한줄에 정렬 (요소의 크기와 상관없이 넘어가면 가고, 사라지면 사라지고)

    wrap : 요소들을 여러줄에 걸쳐서 정렬

    wrap-reverse : 요소들을 여러줄에 걸쳐 정렬 시키는데 반대로 정렬

  10. 예시
    • menu1
    • menu2
    • menu3
    • menu4
    • menu5
    • menu6

  11. flex-flow : 나열순서와 감싸기를 한번에 지정

    flex-direction(나열순서) + flex-wrap(한줄 또는 여러줄)

    두 속성값을 띄워쓰기 한번을 구분해서 작성 (flex-flow: row wrap;) > 왼쪽에서 오른쪽으로 정렬 ,여러줄로 정렬하겠다
