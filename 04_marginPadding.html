margin : 요소의 바깥 여백

단위: px, %, auto(정해진 너비를 없앨 때 사용)

태그 안의 첫번째 자식은 부모요소와 같이 상단 마진값을 가진다

padding : 요소의 안쪽 여백

단위: px, %, auto(정해진 너비를 없앨 때 사용)

마진겹침현상: 마진은 옆으로 또는 위아래로 값이 겹쳐질 경우에 마진값들이 합쳐지는게 아니라 더 큰 쪽의 마진값이 적용됨

margin / padding 값을 주는 방법

margin 이나 padding에 값을 1개 초과해서 부여하면 값을 주는 만큼 방향으로 여백이 생긴다

margin padding : Npx; -> 사방으로 Npx만큼 여백 들어옴

margin padding : Npx Npx; -> 위/아래 , 왼/오른

margin padding : Npx Npx Npx; -> 위 왼 / 오른 아래

margin padding : Npx Npx Npx Npx;-> 위 오른 / 왼 아래

너비값이 정해진 요소를 화면상(띄워진 창의 크기에서) 또는 내 부모요소에서 중간정렬을 위한 값

매번 너비가 바뀌므로 정확하게 수치상으로 계산하는 것은 비효율적, 반응형의 기초 작업의 일부. 많이사용된다

글자만 중간에 보내고 싶다고 박스를 만들지 않고 그냥 사용하면 안된다.
글자 정렬은 다른 속성이 있으므로 무조건 박스생성하여 기본 너비값을 넣어주고 사용할 것

나중에 작성한 여백값이 우선순위로 적용된다

위에 여백값을 주는 방법에서 2개 값을 적어 사용하는 것과 같은 방법으로 사용가능
예를들어 Npx auto라고 입력하면 위/아래에 Npx만큼 여백을 주고, 좌/우로는 가운데 정렬 된다
