position: 요소의 위치 지정방법 유형을 지정

    position 속성값
  1. static : 원래 기존의 상태
  2. relative : absolute 영역 기준점
  3. absolute : 상위요소의 relative를 기준으로 위치를 재배치
    relative가 상위요소(형제태그 아님!)에 오고, absolute가 하위요소에 위치한다
    더 가까운 상위요소에 position속성값을 가진 요소가 있다면, 그 요소를 기준으로 배치된다
  4. fixed : 요소의 위치를 스크린을 기준으로 배치
  5. sticky : 필수적으로 top, bottom, left, right 중에 하나를 설정해 주어야 하고,
    설정된 위치에 도달전까지는 static 처럼 행동하다가 위치에 다다르면 fixed처럼 행동
  6. relative가 없이 absolute단독으로 사용될 경우, body를 기준으로 배치된다(기준점이 없이 배치가 붕 떠서 상단으로 올라감)
  7. position과 함께 사용되는 속성
    • top, bottom, left, right
    • relative된 기준점으로 부터의 위치값을 지정하는 속성
    • 단위: px, %, auto 0
    • 단위를 음수로 사용도 가능하다
  8. span이나 a태그 같은 인라인요소에 absolute값을 주게되면 블락속성을 가지게 된다
